    将列表存储为csv文件

    python
    创建时间:2020-04-30 21:58
    阅读:
    1. 主体
    2. 备注
      1. 查询当前工作路径
      2. 参数
      3. 中文

    主体

    import pandas as pd
list = [[1,2,3],[4,5,6],[7,8,9]]
name = ['id', 'uid', 'time']
test = pd.DataFrame(columns=name, data=list)
test.to_csv('D:/test.csv')

    打开生成的csv文件，可以看到我们想要的数据

    备注

    查询当前工作路径

    import os
os.getcwd() #获取当前工作路径

    参数

    分隔符sep:Field delimiter for the output file(default “,”)

    test.to_csv('D:/test.csv',sep='?') #使用?分隔需要保存的数据，如果不写，默认是 ','

    替换空值 na_rep: A string representation of a missing value (default ‘ ‘)

    test.to_csv('D:/test.csv',na_rep='NA') #确实值保存为NA，如果不写，默认是空

    格式 float_format: Format string for floating point numbers

    test.to_csv('D:/test.csv',float_format='%.2f') #保留两位小数

    是否保留列名 header: Whether to write out the column names (default True)

    test.to_csv('D:/test.csv',header=0) #不保存列名

    是否保留行索引 index: whether to write row (index) names (default True)

    test.to_csv('D:/test.csv',index=0) #不保存行索引
